## 它做什么

- 通过自然语言对话搭建全栈 Web 应用，过程中实时预览
- 底层技术栈全包：内置云端后端，数据库和鉴权无需配置
- 一键发布到托管 URL 或自定义域名；代码随时可导出到 GitHub

## 值得注意的细节

- 主界面是跑起来的应用，不是代码——编辑靠指着说
- 所有方案都不收席位费——协作者不限量；商业化纯靠用量积分
- 项目始终是真实代码库：随时可以导出到 GitHub，在 IDE 里继续开发

## 适合谁

想验证一个真实能跑的想法、又不想等工程排期的产品人、设计师和创始人。
