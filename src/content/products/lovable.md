## What it does

- Builds full-stack web apps from natural-language conversation, with a live preview as you go
- Handles the stack underneath: built-in cloud backend for database and auth, no setup
- One-click publish to a hosted URL or custom domain; code stays exportable to GitHub

## Details worth noting

- The primary interface is the running app, not the code — editing happens by pointing and describing
- No seat fees on any plan — unlimited collaborators; monetization is purely usage credits
- Projects remain real codebases: you can eject to GitHub and continue in an IDE anytime

## Who it's for

Product people, designers, and founders who want to test a real, working idea without waiting for engineering time.
