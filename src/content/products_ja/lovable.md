## できること

- 自然言語の対話からフルスタックの Web アプリを構築。進めながらライブプレビューで確認できる
- 裏側のスタックはお任せ。データベースと認証はビルトインのクラウドバックエンドで、セットアップ不要
- ワンクリックでホスティング URL やカスタムドメインに公開。コードはいつでも GitHub にエクスポートできる

## 注目したいディテール

- 主役のインターフェースはコードではなく、動いているアプリ — 指差して説明するように編集する
- どのプランにもシート課金がなく、コラボレーターは無制限。収益化は純粋に利用クレジットのみ
- プロジェクトは本物のコードベースのまま。いつでも GitHub に出して、IDE で続きを書ける

## 向いている人

エンジニアリングの順番待ちをせずに、本当に動くアイデアを試したいプロダクト担当者、デザイナー、創業者。
